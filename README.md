# HOMEGROUND

A mobile-first cricket scorecard management app by **Sanjay Yadav**.

## Features

- **Match Management**: Create Single Matches, Series, and Tournaments.
- **Live Scoring**: Ball-by-ball scoring interface with extras, wickets, and undo functionality.
- **Scorecards**: Detailed match statistics and summaries.
- **Players & Teams**: Manage team compositions and player roles.
- **Mobile First**: Designed for local cricket scorers on the go.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/app`: Page routes (Home, Match Creation, Live Match).
- `src/components`: UI components (ScoringBoard, MatchHeader, etc.).
- `src/contexts`: State management (AppContext).
- `src/lib`: Utilities.
